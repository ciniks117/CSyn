Syntax

Param = param_stirredTankReactor_traj2()

Description

Parameters for the stirred-tank reactor benchmark. The parameters include input constraints, disturbances as well as the parameters of the motion primitive like initial set, goal state, etc..

Output Arguments

Param

a structure containing following options

.R0

initial set (class: interval)

.xf

goal state

.U

set of admissible control inputs (class:; interval)

.W

set of uncertain disturbances (class: interval)
